# Neovim config based on LazyNvim


## Installation 
Create backup and clone repo
```bash
# Make backup
mv ~/.config/nvim{,.bak}
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
# Clone repo
git clone git@github.com:matheus-mcosta/nvim.git ~/.config/nvim

```
Check installation inside Neovim
```vim
:checkhealth
```
